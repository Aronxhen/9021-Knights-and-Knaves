# 9021 Knights and Knaves logic problems

### 项目介绍：
解决骑士与骗子逻辑问题，通过读取输入文件中的陈述，判断每个人的身份（骑士或骗子）。该算法可以用于数据分析中的复杂规则判定，例如，根据一组规则来判断客户行为模式、分类或异常检测。

### 项目组成：
- assignment1.pdf （项目介绍）
- knights_and_knaves.py （项目源代码）
- 其他测试文件
